# 英语启蒙产品 · 开发工作清单

> **项目基调**：0成本启动 · 极简去角色化 · 星星+涂色/绘本激励 · 覆盖幼儿园全学段(L1-L6)
> **MVP目标**：完成 L1 首期（20集/1个月），可在 iPad 浏览器中完整体验的互动学习闭环

---

## 一、盘前准备（开发前必须完成）

### 1.1 首期核心内容包制作
- [ ] 确定 MVP 首期 20 集的主题分组（4个Unit × 5集）
- [ ] 整理首期核心词汇表（约30-40词），按 Unit 分组，输出为 JSON 格式
- [ ] 整理首期核心短句（约5-10句），对应到每个 Unit
- [ ] 使用 Edge-TTS 批量生成所有单词和短句的高质量 MP3 音频文件
- [ ] 为每个单词准备对应的插画素材（可用 AI 绘图或开源素材）

### 1.2 UI/UX 高保真设计
- [ ] 设计主城学习地图页（极简风格：星星总数 + 关卡气泡排列）
- [ ] 设计互动学习页（场景构图、点击物品放大发光的视觉效果）
- [ ] 设计涂色奖励页（线稿样式、调色盘布局）
- [ ] 设计绘本阅读页（翻页交互、图文排版）
- [ ] 设计家长控制中心页（数据面板、防沉迷设置）
- [ ] 确定全局设计规范（护眼色系色值表、字体字号、按钮尺寸、圆角标准）

### 1.3 交互动效定义
- [ ] 定义"待点击物品"的呼吸动画参数（频率、幅度）
- [ ] 定义"点击反馈"的动效（缩放、发光、弹跳）
- [ ] 定义"星星飞入"的过关庆祝动效
- [ ] 定义"涂色完成"和"绘本解锁"的阶段庆祝动效
- [ ] 定义页面间转场动画（淡入淡出 / 滑动）

### 1.4 本地数据结构设计
- [ ] 设计 localStorage 的 JSON 数据模型（星星数、已解锁关卡、学习日期、防沉迷计时等）
- [ ] 设计"学习进度一键导出/恢复"的机制方案（应对清缓存/换设备）
- [ ] 明确防沉迷逻辑（每日首次打开记录时间戳，累计超过15分钟触发锁屏）

---

## 二、正式开发（前端工程）

### 2.1 基础工程搭建
- [ ] 确定最终技术栈（纯HTML5/CSS3/JS 或 Vue/React + Vite）
- [ ] 初始化项目目录结构与构建工具
- [ ] 实现全局样式系统（护眼色系、响应式布局、iPad横屏适配）
- [ ] 实现音频播放管理器（预加载 MP3、播放队列控制）
- [ ] 实现 localStorage 数据管理模块（读写、校验、防沉迷计时）

### 2.2 主城地图页
- [ ] 开发学习地图（关卡气泡：已完成/当前/锁定 三态）
- [ ] 星星总数展示区
- [ ] 家长中心入口（带算术验证锁）
- [ ] 关卡点击跳转至对应学习页

### 2.3 互动学习页（核心）
- [ ] 场景渲染（背景 + 可点击物品布局）
- [ ] 点读交互逻辑（点击 → 发光动画 → 播放MP3 → 显示单词）
- [ ] 短句展示与发音（第二轮点击触发）
- [ ] 星星进度条（本集需集齐N颗星才能过关）
- [ ] 通关结算弹窗（星星飞入动画 + 跳转涂色/绘本奖励 或 回主页）

### 2.4 涂色奖励页
- [ ] 线稿渲染（根据当前 Unit 主题动态加载）
- [ ] 调色盘与填色交互
- [ ] 涂色完成后保存到"我的画廊"（localStorage）
- [ ] 返回主页入口

### 2.5 绘本阅读页
- [ ] 绘本翻页交互（左右滑动 / 点击翻页）
- [ ] 图文混排渲染（插画 + 英文短句）
- [ ] 点击句子播放朗读音频
- [ ] 阅读完成标记

### 2.6 家长控制中心
- [ ] 今日学习数据展示（时长、完成集数、获得星星）
- [ ] 已掌握词汇列表（按 Unit 分组）
- [ ] 防沉迷时长设置（15/20/30分钟可选）
- [ ] 学习报告导出（可打印的HTML/PDF页面）

## 三、测试与优化

### 3.1 功能测试
- [ ] 主流程串联测试（主页 → 学习 → 涂色/绘本 → 回主页）
- [ ] 防沉迷锁屏触发测试
- [ ] localStorage 数据持久化测试（关闭/刷新后进度不丢失）
- [ ] 家长验证锁测试

### 3.2 设备适配测试
- [ ] iPad Safari 横屏全屏测试
- [ ] iPad Chrome 测试
- [ ] Android 平板测试
- [ ] 投屏到电视/电脑测试（AirPlay / 有线）

### 3.3 体验优化
- [ ] 触控热区优化（按钮面积 ≥ 44×44pt，适配幼儿手指）
- [ ] 动画流畅度优化（确保60fps，消除卡顿）
- [ ] 音频预加载优化（消除点击后的延迟静默）
- [ ] 护眼色系最终校准（实机查看色差）
