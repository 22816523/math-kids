## 1. 全局设计规范

### 1.1 护眼色系（色值表）
| 用途 | 色名 | Hex值 | 说明 |
|------|------|-------|------|
| 主背景 | 燕麦暖白 | `#FDF6E3` | 大面积底色，低蓝光柔和 |
| 关卡气泡（活跃）| 暖阳黄 | `#FFD166` | 高辨识的可点击状态 |
| 关卡气泡（锁定）| 灰黄 | `#EAE2B7` | 低饱和度锁定态 |
| 正确反馈 | 薄荷绿 | `#A8E6CF` | 统一的"正确/成功"色 |
| 警告/错误 | 珊瑚粉 | `#FF595E` | 提示色，仅用于家长端 |
| 主文字 | 深咖啡 | `#5C4A4A` | 不刺眼的深色文字 |
| 辅助文字 | 暖灰 | `#999999` | 次要信息 |
| 白色面板 | 纯白 | `#FFFFFF` | 卡片/面板背景 |
| 阴影 | 咖啡透明 | `rgba(92,74,74,0.15)` | 统一阴影色 |

### 1.2 字体与圆角
- **字体**：`'Comic Sans MS', 'Nunito', 'PingFang SC', sans-serif`
- **主标题**：36px / bold
- **内容文字**：24px / regular
- **按钮文字**：28px / bold
- **圆角标准**：小元素 `16px`，卡片 `30px`，全圆按钮 `50px`
- **最小触控热区**：`60×60px`（超过 Apple HIG 的 44pt 标准，适配幼儿手指）

## 2. 交互动效参数表

### 2.1 待点击物品 · 呼吸动画
```css
@keyframes breathe {
    0%, 100% { transform: scale(1); }
    50% { transform: scale(1.08); }
}
/* 频率: 2s 一个周期, 无限循环, 缓动函数: ease-in-out */
```

### 2.2 点击反馈 · 三段式
| 阶段 | 时长 | 效果 |
|------|------|------|
| 按下瞬间 | 0ms | `scale(0.9)` 内缩 |
| 弹起放大 | 200ms | `scale(1.2)` + `box-shadow 发光` |
| 恢复原位 | 300ms | `scale(1.0)` 回弹 |

### 2.3 星星飞入动画
- 星星从物品位置飞向顶部星星栏，路径为抛物线
- 持续时间：`600ms`
- 缓动曲线：`cubic-bezier(0.175, 0.885, 0.32, 1.275)`（overshooting弹跳）
- 到达后闪烁一次金光

### 2.4 页面转场
- 统一使用 `fade + slide` 组合：
  - 进入：从右滑入 `translateX(100%) → 0`，同时淡入 `opacity: 0 → 1`
  - 退出：向左滑出 `translateX(0) → -30%`，同时淡出 `opacity: 1 → 0`
  - 时长：`400ms`

### 2.5 通关庆祝
- 全屏半透明遮罩 `rgba(255,255,255,0.8)` + `backdrop-filter: blur(8px)`
- 中央弹出大号文字（如 "Awesome!"）+ 彩色粒子喷发
- 持续 `2s` 后自动显示"下一步"按钮

## 3. localStorage 数据模型

### 3.1 数据结构
```json
{
  "version": "1.0",
  "user": {
    "stars_earned": 0,
    "stars_spent": 0,
    "current_unit": 1,
    "current_lesson": 1
  },
  "lessons": {
    "1": { "completed": false, "stars": 0, "date": null },
    "2": { "completed": false, "stars": 0, "date": null }
  },
  "units": {
    "1": { "completed": false },
    "2": { "completed": false },
    "3": { "completed": false },
    "4": { "completed": false }
  },
  "shop": [
    { "id": "color_fruit", "type": "coloring", "title": "水果拼盘涂色卡", "cost": 10, "unit": 1 },
    { "id": "color_animal", "type": "coloring", "title": "动物园涂色卡", "cost": 10, "unit": 2 },
    { "id": "book_rainbow", "type": "book", "title": "彩虹绘本", "cost": 15, "unit": 3 },
    { "id": "book_body", "type": "book", "title": "身体绘本", "cost": 15, "unit": 4 }
  ],
  "my_collection": [],
  "anti_addiction": {
    "today_date": "2026-03-10",
    "today_minutes": 0,
    "daily_limit_minutes": 15,
    "is_locked": false
  }
}
```

**关键变更说明**：
- `stars_earned` 与 `stars_spent` 分开记录，可用余额 = earned - spent
- `shop` 定义可兑换的商品及所需星星数
- `my_collection` 存储已兑换物品的 id 列表，代表永久拥有

### 3.2 存储 Key
- Key 名：`english_learning_mvp`
- 读取：`JSON.parse(localStorage.getItem('english_learning_mvp'))`
- 写入：`localStorage.setItem('english_learning_mvp', JSON.stringify(data))`

### 3.3 防沉迷逻辑
1. **每次打开 App / 页面跳转**时，读取 `anti_addiction.today_date`。
2. 若日期不是今天 → 重置 `today_minutes = 0`，更新日期。
3. 每 `60s` 通过 `setInterval` 自增 `today_minutes += 1`。
4. 当 `today_minutes >= daily_limit_minutes`：
   - 立即弹出全屏锁定遮罩："今天学习时间到啦，明天再来吧！"
   - 设置 `is_locked = true`，禁用所有交互。
   - 仅保留"家长解锁"按钮（需通过算术验证）。

### 3.4 进度导出/恢复
- **导出**：将整个 JSON 序列化为 Base64 字符串，显示为一个二维码或可复制的文本码。
- **恢复**：在家长中心提供"粘贴恢复码"入口，反序列化后覆盖 localStorage。
