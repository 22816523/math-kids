# 数学启蒙项目上下文

## 1. 项目概况

- 项目名称：数学启蒙 MVP Web
- 所在目录：`888-自定义产品/3-数学启蒙/mvp_web`
- 主要页面：`math.html`、`baisu.html`、`shape.html`、`measure.html`
- 目标用户：低龄儿童，当前默认按“不识字，需要语音引导”的交互标准设计

## 2. 核心业务目标

- 让孩子可以靠语音提示独立完成基础数学启蒙练习
- 保持练习流顺滑，答对后自动进入下一题，减少等待和额外点击
- 在手机与 Pad 上都保证拖拽交互稳定，手指轨迹与元素显示一致

## 3. 关键规则与约束

- 练习题题干默认需要自动播报，并提供可点击的喇叭按钮重播
- 交互优先服务儿童直觉，不依赖复杂文字说明
- 触屏场景要优先考虑 iPad / Android Pad 的定位和视口差异
- 当前工程是纯 HTML/CSS/JS，无打包器

## 4. 代码结构

- `mvp_web/js/practice-support.js`：题干播报/重播与拖拽定位通用能力
- `mvp_web/js/baisu-board.js`：百数板相关练习
- `mvp_web/js/shape.js`：图形认知、分类、找一找
- `mvp_web/js/measure.js`：钟表、比较、长度、重量、人民币
- `mvp_web/js/math.js`：加减法练习
