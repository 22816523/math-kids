# 校校读吧 AI伴读V1.0 项目上下文

## 1. 项目概述

"校校读吧"平台由三个端组成：管理平台（运营人员）、老师端（小程序）、学生端（小程序）

核心业务：运营方在管理平台创建阅读计划 → 老师为班级报名 → 学生完成每日阅读任务打卡

## 3. 重要业务决策

### 2026-03-02：阅读计划按年级配置不同书目
- 变更前：一个阅读计划下所有年级共用同一套书目
- 变更后：一个阅读计划下，每个年级配置不同的书目（如3年级读3本、4年级读4本）
- 影响范围：三端PRD均需调整，老师端原型已加年级Tab
