常见问题
如何创建阅读计划
→
1. 点击首页"创建计划"按钮
2. 填写计划基本信息（名称、时间等）
3. 选择或搜索书籍
4. 划分章节和设置打卡要求
5. 预览并发布计划
如何批改学生测评
→
1. 进入"测评管理"页面
2. 选择需要批改的测评
3. 查看学生答题情况
4. 对主观题进行评分和点评
5. 保存批改结果
如何查看班级数据报告
→
1. 进入"数据报告"页面
2. 选择要查看的班级
3. 查看打卡率、测评成绩等数据
4. 可以导出数据报告
如何添加学生到班级
→
1. 进入"班级管理"页面
2. 选择要添加学生的班级
3. 点击"学生管理"
4. 通过邀请码或扫码方式添加学生
如何与家长沟通
→
1. 进入"家长沟通"页面
2. 选择要沟通的学生家长
3. 发送消息或查看历史沟通记录
4. 可以群发通知给所有家长
