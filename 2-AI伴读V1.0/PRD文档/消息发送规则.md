## 4. 消息发送频率控制

### 4.1 自动触发消息

| 消息类型                   | 发送频率 | 说明                       |
| -------------------------- | -------- | -------------------------- |
| 今日阅读任务未完成提醒     | 每日1次  | 每日21:00统一发送给老师    |
| 学生今日阅读任务未完成提醒 | 每日1次  | 每日20:00统一发送给学生    |
| 学生今日测评任务未完成提醒 | 每日1次  | 每日20:30统一发送给学生    |
| 学生完成阅读任务通知       | 实时     | 学生打卡后立即发送         |
| 学生完成测评通知           | 实时     | 学生提交测评后立即发送     |
| 新阅读计划发布通知         | 按需     | 平台发布新计划时发送       |
| 阅读计划开放报名通知       | 按需     | 阅读计划开放报名时发送     |
| 学生加入班级通知           | 实时     | 学生加入班级后立即发送     |
| 学生被移除班级通知         | 实时     | 老师移除学生后立即发送     |
| 班级报名成功通知           | 实时     | 老师完成班级报名后立即发送 |

### 4.2 手动发送消息

| 消息类型         | 发送频率 | 说明                   |
| ---------------- | -------- | ---------------------- |
| 系统功能更新通知 | 按需     | 管理员根据更新情况发送 |
| 平台升级维护通知 | 按需     | 管理员在维护前提前发送 |

## 5. 消息免打扰设置

用户可在个人中心设置消息接收偏好：

- 关闭/开启微信消息推送
- 关闭/开启小程序消息推送
- 设置免打扰时间段

## 6. 注意事项

1. **用户授权**

   - 小程序订阅消息需要用户主动授权
   - 微信消息需要用户关注公众号
   - 未授权用户无法接收对应类型的消息
2. **消息频率**

   - 避免频繁发送消息，影响用户体验
   - 同一用户同一类型消息每日最多发送3次
3. **隐私保护**

   - 消息内容不得泄露用户隐私
   - 学生姓名等敏感信息需脱敏处理
   - 遵守《个人信息保护法》相关规定
