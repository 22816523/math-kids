## 3. 消息变量说明

| 变量名     | 说明                           | 示例                   |
| ---------- | ------------------------------ | ---------------------- |
| {班级名称} | 班级的完整名称                 | 三年级(1)班            |
| {人数}     | 学生数量                       | 5                      |
| {学生名单} | 学生姓名列表                   | 张三、李四、王五       |
| {学生姓名} | 单个学生的姓名                 | 张三                   |
| {书名}     | 书籍名称（展示时统一加书名号） | 小王子                 |
| {任务内容} | 具体的阅读任务内容             | 阅读第1-3章            |
| {分数}     | 测评得分                       | 85                     |
| {计划名称} | 阅读计划的名称                 | 2025春季阅读计划       |
| {书目列表} | 书籍名称列表                   | 《小王子》《夏洛的网》 |
| {数量}     | 书籍数量                       | 10                     |
| {开始日期} | 计划开始日期                   | 2025年3月1日           |
| {结束日期} | 计划结束日期                   | 2025年6月30日          |

## 4. 注意事项

1. **用户授权**

   - 小程序订阅消息需要用户主动授权
   - 未授权用户无法接收对应类型的消息
2. **隐私保护**

   - 消息内容不得泄露用户隐私
   - 学生姓名等敏感信息需按平台规则处理
