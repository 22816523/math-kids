# 客户管理系统V1.1 新增功能原型

## 📋 概述

本目录包含客户管理系统从V1.0升级到V1.1版本新增的8个核心功能的界面原型

## 🎯 核心目标

**"一个学校，一个档案；一个老师，一个档案"**

实现客户和联系人数据的唯一性、标准化和跨系统打通，建立公司级统一数据资产管理体系

## 📁 文件说明

### 导航页面
- **index.html** - 新增功能导航首页，展示所有8个新增功能的入口

### 功能原型页面

#### 1. 主客户管理
- **客户唯一标识.html** - 为每个主客户生成全局唯一ID，实现跨系统识别和匹配

#### 2. 联系人管理
- **联系人唯一标识.html** - 建立公司级统一联系人数据库，以学校+姓名+手机号作为唯一标识
- **归属权管理.html** - 支持联系人的分配、转移、回收等操作，实现公司级资产管理
- **全局搜索与权限控制.html** - 提供公司级联系人搜索功能，实现分级权限控制

#### 3. 系统集成
- **跨系统数据打通.html** - 打通智慧九科星、中课云、研究院稿费报销等系统
- **数据同步日志.html** - 实时监控各系统间的数据同步状态，记录同步日志

#### 4. 数据安全
- **敏感信息保护配置.html** - 配置敏感信息脱敏规则，保护客户隐私数据

#### 5. 报表分析
- **统计报表.html** - 生成多维度统计报表，可视化展示数据全景

### 样式文件
- **common.css** - 通用样式文件，定义了所有页面的统一视觉风格

## 🎨 设计特点

1. **风格统一** - 所有页面采用统一的视觉风格，与V1.0版本保持一致
2. **响应式设计** - 支持不同屏幕尺寸的自适应显示
3. **交互友好** - 清晰的导航结构，直观的操作界面
4. **数据可视化** - 使用卡片、表格、统计图表等多种形式展示数据

## 🚀 使用方法

1. 在浏览器中打开 `index.html` 查看导航首页
2. 点击功能卡片进入对应的详细原型页面
3. 所有页面均为静态展示，用于需求评审和设计参考

## 📝 注意事项

- 本原型为静态HTML页面，不包含实际的业务逻辑和数据交互
- 页面中的数据均为示例数据，用于展示界面效果
- 图表部分使用占位符展示，实际开发时需接入图表库（如ECharts、Chart.js等）
- 所有页面使用UTF-8编码，确保中文正常显示
