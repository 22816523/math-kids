## 1. 背景与目标

### 1.1 项目背景

需要一个本地使用的家庭记账工具，用于记录家庭支出。用户不希望支付云服务费用，因此采用 Windows 安装包形式，数据保存在本地，并通过手动备份文件支持换电脑后的数据恢复。

### 1.2 产品目标

- 支持家庭支出记录的手工新增、Excel 导入、查询、编辑、删除。
- 支持按自然月、自然年、分类、成员查看统计报表。
- 支持柱状图、饼状图、折线图和统计表格，帮助了解家庭支出分布。
- 支持本地数据存储、备份导出、从备份恢复。
- 支持 6 位数字密码保护账本。

### 1.3 项目范围

- Windows 安装包。
- 本地账本数据存储。
- 6 位数字密码进入。
- 首次新建账本与从备份恢复。
- 仪表盘。
- 记账明细。
- 手工新增。
- Excel 导入。
- 报表。
- 基础设置。
- 数据管理。

## 2. 用户分析

### 2.1 目标用户与场景

目标用户为家庭记账使用者。核心场景包括：日常记录家庭支出、定期整理 Excel 账单导入、查看家庭支出分布、按成员或分类分析支出、换电脑时通过备份文件恢复历史数据。

- 信用卡

- 微信
- 现金
- 京东支付
- 美团支付
- 拼多多
- 云闪付
- 其他

### 3.4 基础选项维护规则

- 支持维护分类、成员、来源平台。
- 初始值使用已确认的分类、成员、来源平台选项。
- 新增或修改时不允许重名。
- 允许删除或调整选项，但不影响历史记录原值。
- 修改选项只影响后续新记录，历史记录保持原值。
- 基础设置操作成功后显示成功提示，并刷新当前列表。

## 4. 数据字段与校验

### 4.1 记账字段

| 字段 | 规则 |
| ---- | ---- |
| 日期 | 必填，格式为 YYYY-MM-DD |
| 金额 | 必填，正数，保留 2 位小数 |
| 分类 | 必填，必须从已有分类选项中选择 |
| 成员 | 必填，必须从已有成员选项中选择 |
| 备注 | 可为空 |
| 来源平台 | 必填，必须从已有来源平台选项中选择 |

### 4.2 重复判定

当日期、金额、分类、成员、备注、来源平台全部相同时，判定为重复记录。重复记录提示并拦截。

## 6. 功能详情

### 6.1 首次启动与日常启动

#### 功能说明

首次使用时，系统需要创建账本或从备份文件恢复账本。日常使用时，用户输入 6 位数字密码后进入仪表盘。

#### 业务规则

- 首次启动显示新建账本、从备份恢复。
- 日常启动输入 6 位数字密码后进入仪表盘。
- 密码错误次数不限制。
- 支持修改数字密码。
- 修改密码时需要先输入旧密码。
- 首次设置和修改密码成功后显示成功提示。
- 忘记密码时，通过备份文件恢复账本并重新设置数字密码。

#### 异常处理

- 备份文件不可用或解析失败时，提示恢复失败。

### 6.2 仪表盘

#### 功能说明

仪表盘作为日常启动后的默认页面，用于快速查看当前家庭支出概况。

#### 业务规则

- 启动页为仪表盘。
- 统计口径按当前系统日期所在自然月、自然年统计。
- 展示本月总支出、本年累计支出、各分类占比、各成员支出、本月支出趋势。
- 本月支出趋势使用折线图。
- 显示最近 5 条支出。
- 点击图表或卡片可跳转到对应报表页。

#### 异常处理

- 无数据时显示暂无数据。

### 6.3 记账明细

#### 功能说明

用于查看、筛选、搜索、编辑、删除记账记录。

#### 业务规则

- 列表字段：日期、金额、分类、成员、来源平台、备注、操作。
- 操作栏包含编辑、删除。
- 默认按日期倒序显示。
- 支持按日期范围、成员、来源平台、分类、金额范围筛选。
- 支持备注关键词搜索，并高亮命中关键词。
- 支持分页，每页 50 条。
- 分页方式为上一页、下一页、页码跳转。
- 不支持批量删除。
- Excel 明细导出暂不需要。

#### 异常处理

- 无记录时显示暂无记录。
- 删除时需要二次确认。
- 删除成功后显示删除成功，并刷新当前列表。

### 6.4 手工新增

#### 功能说明

用于在软件内手工新增单条支出记录，与 Excel 导入分开。

#### 业务规则

- 字段与导入字段一致：日期、金额、分类、成员、备注、来源平台。
- 保存成功后自动清空表单。
- 保存成功后显示保存成功。

#### 异常处理

- 必填项缺失时不允许保存。
- 金额不是正数时不允许保存。
- 分类、成员、来源平台不存在时不允许保存。

### 6.5 编辑记录

#### 功能说明

用于修改已存在的记账记录。

#### 业务规则

- 编辑时默认带出原值。
- 保存成功后显示保存成功，并返回记账明细列表。

#### 异常处理

- 校验规则与手工新增一致。

### 6.6 Excel 导入

#### 功能说明

用于导入手动整理后的 Excel 账单文件。Excel 内已经人工分好类，不需要映射规则。

#### 业务规则

- 支持下载 Excel 模板。
- 模板列顺序固定为：日期、金额、分类、成员、备注、来源平台。
- 按系统导出的模板读取，首行为表头，从第二行开始读取数据。
- 列名或列顺序不符合模板时，整批不能导入。
- 空白行不自动忽略，按错误处理。
- Excel 中不存在的分类、成员、来源平台算错误。
- 日期、金额、分类缺失时拒绝导入。
- 存在错误时整批不导入。
- 导入批次不限。
- 导入页面流程：下载模板、选择 Excel 文件、系统校验、显示校验结果、无错误时确认导入、有错误时下载错误文件且不允许导入。
- 导入预览显示总行数、可导入行数、错误行数、重复行数。
- 支持下载错误文件，错误原因显示在对应行最后一列。
- 导入结果只有全部成功或全部失败。
- 成功时显示成功导入条数和导入时间。
- 失败时提示下载错误文件。

#### 异常处理

- 重复记录提示并拦截。
- 模板错误、字段错误、数据错误、重复记录均导致整批不能导入。

### 6.7 报表

#### 功能说明

用于查看家庭支出统计，支持按月、按年、按分类、按成员统计。

#### 业务规则

- 支持按月、按年、按分类、按成员。
- 按月统计口径：每个自然月生成一行统计数据。
- 按年统计口径：每个自然年生成一行统计数据。
- 按月/按年统计字段：统计周期、各分类支出金额、总支出金额。
- 按分类统计字段：分类、支出金额。
- 按成员统计字段：成员、支出金额。
- 支持柱状图、饼状图。
- 按月、按年、按分类、按成员均支持柱状图和饼状图。
- 支持统计表格。
- 支持时间范围筛选。
- 支持按分类、成员、来源平台筛选。
- 支持报表统计结果导出 Excel。
- 历史旧名称按原值单独统计。
- 统计金额为 0 的分类不展示。

#### 异常处理

- 报表无数据时显示暂无数据。
- 图表无数据时显示暂无数据，不显示空图。

### 6.8 数据管理

#### 功能说明

用于管理本地数据、备份、恢复和密码。

#### 业务规则

- 数据默认目录：文档\家庭记账\数据。
- 支持打开数据目录。
- 数据管理功能包括导出备份、从备份恢复、打开数据目录、修改数字密码。
- 备份方式为导出系统专用文件，换电脑后导入恢复。
- 备份导出位置由用户自行选择。
- 备份文件名由系统自动生成，格式为家庭记账_年月日_时分。
- 导入备份会覆盖当前数据，必须二次确认。
- 导入备份前显示备份时间、记录数量、备份文件名。
- 备份恢复后重新设置 6 位数字密码。
- 导出备份、从备份恢复均显示成功或失败结果提示。
- 自动提醒备份暂不需要。

#### 异常处理

- 备份文件损坏、格式不匹配或读取失败时，提示恢复失败。

## 7. 非功能需求

### 7.1 平台

- 仅支持 Windows 版安装包。

### 7.2 存储

- 数据本地保存。
- 不使用云端同步。
- 换电脑通过备份文件恢复数据。

### 7.3 安全

- 使用 6 位数字密码进入账本。
- 不限制密码错误次数。
- 忘记密码不支持找回原密码，只支持通过备份文件恢复账本并重新设置密码。

## 8. 风险提示

- 本地存储依赖用户自行保管数据和备份文件，电脑损坏且无备份时可能丢失历史数据。
- 备份文件需要妥善保存，否则可能被他人恢复账本数据。
- 基础选项修改后历史记录保留旧值，报表中可能同时出现新旧名称。
- Excel 导入采用整批成功或失败规则，任一错误都会导致整批不能导入。
