## 需求设计记录

- 创建时间：2026-04-27

### 待确认问题

- 终端偏好：安装包优先
- 平台：Windows 版
- 导入方式：文件导入
- 导入格式：Excel
- 导入来源：手动整理后的 Excel
- Excel 模板列顺序：日期、金额、分类、成员、备注、来源平台
- Excel 模板下载：支持
- Excel 读取规则：按系统导出的模板读取，首行为表头，从第二行开始读取数据
- Excel 模板校验：列名或列顺序不符合模板时，整批不能导入
- Excel 导入页面流程：下载模板、选择 Excel 文件、系统校验、显示校验结果、无错误时确认导入、有错误时下载错误文件且不允许导入
- 分类处理：Excel 内已分好类，不需要映射规则
- 分类来源：人工分好
- 录入字段：日期、金额、分类、成员、备注、来源平台
- 备注规则：可为空
- 来源平台规则：必填
- 成员规则：必填
- 分类规则：必须从已有分类选项中选择；Excel 中不存在的分类算错误
- 成员和来源平台规则：Excel 中不存在的成员或来源平台算错误
- 空白行处理：不自动忽略，按错误处理
- 日期格式：YYYY-MM-DD
- 导入校验：日期、金额、分类缺失时拒绝导入
- 金额规则：全部为正数，只记录家庭支出
- 金额精度：保留 2 位小数
- Excel 导入预览：显示总行数、可导入行数、错误行数、重复行数
- Excel 错误文件下载：支持，错误原因显示在对应行最后一列
- Excel 导入规则：存在错误时整批不导入
- Excel 导入结果：全部成功或全部失败；成功时显示成功导入条数和导入时间；失败时提示下载错误文件
- 重复校验：提示重复并拦截
- 重复判定：日期、金额、分类、成员、备注、来源平台都相同才算重复
- 报表维度：按月、按年、按分类、按成员
- 报表时间维度：支持按月、按年
- 按月统计口径：每个自然月生成一行统计数据
- 按年统计口径：每个自然年生成一行统计数据
- 按月/按年统计字段：统计周期、各分类支出金额、总支出金额
- 按分类统计字段：分类、支出金额
- 按成员统计字段：成员、支出金额
- 报表图表：柱状图、饼状图
- 图表维度：按月、按年、按分类、按成员均支持柱状图和饼状图
- 报表展示：表格
- 报表时间范围筛选：需要
- 报表筛选条件：分类、成员、来源平台
- 统计表格：需要
- 报表统计结果导出 Excel：支持
- 报表空状态：暂无数据
- 图表空状态：暂无数据，不显示空图
- 导出备份：需要
- 备份方式：导出系统专用文件，换电脑后导入恢复
- 自动提醒备份：暂不需要
- 导入批次：不限
- 日期范围筛选：需要
- 备注关键词搜索：需要
- 手工新增：需要，且与导入分开
- 手工新增字段：与导入字段一致
- 手工新增保存后：自动清空表单
- 手工新增成功反馈：显示“保存成功”
- 记录操作：支持编辑、删除
- 删除确认：需要
- 编辑时：默认带出原值
- 编辑保存成功反馈：显示“保存成功”，返回记账明细列表
- 删除成功反馈：显示“删除成功”，刷新当前列表
- 批量删除：不需要
- 列表排序：默认按日期倒序
- 记账明细列表字段：日期、金额、分类、成员、来源平台、备注、操作
- 记账明细操作栏：编辑、删除
- 成员筛选：需要
- 来源平台筛选：需要
- 分类筛选：需要
- 金额范围筛选：需要
- 备注关键词高亮：需要
- 分页：需要
- 每页条数：50
- 分页方式：上一页、下一页、页码跳转
- 记账明细空状态：暂无记录
- Excel 明细导出：暂不需要
- 启动页：仪表盘
- 仪表盘内容：本月总支出、本年累计支出、各分类占比、各成员支出、本月支出趋势
- 仪表盘跳转：点击图表或卡片可跳转到对应报表页
- 仪表盘统计口径：按当前系统日期所在自然月、自然年统计
- 仪表盘最近记录：显示最近 5 条支出
- 本月支出趋势：折线图
- 进入方式：数字密码
- 数字密码位数：6 位
- 从备份恢复：需要
- 忘记密码处理：通过备份文件恢复账本并重新设置数字密码
- 首次启动：显示新建账本、从备份恢复
- 日常启动：输入 6 位数字密码后进入仪表盘
- 密码错误次数限制：不需要
- 修改数字密码：支持
- 修改密码验证：需要先输入旧密码
- 页面菜单：仪表盘、记账明细、手工新增、Excel 导入、报表、基础设置、数据管理
- 数据默认目录：文档\家庭记账\数据
- 打开数据目录：需要
- 备份导出位置：用户自行选择
- 备份文件命名：系统自动生成，格式为 家庭记账_年月日_时分
- 导入备份覆盖确认：需要
- 导入备份预览：显示备份时间、记录数量、备份文件名
- 备份恢复后密码：重新设置 6 位数字密码
- 数据管理功能：导出备份、从备份恢复、打开数据目录、修改数字密码
- 来源平台选项：信用卡、储蓄卡、支付宝、微信、现金、京东支付、美团支付、拼多多、云闪付、其他
- 基础选项维护：需要，支持维护分类、成员、来源平台
- 基础选项初始值：使用已确认的分类、成员、来源平台选项
- 基础设置功能：分类维护、成员维护、来源平台维护
- 基础选项重名规则：新增或修改时不允许重名
- 基础选项删除规则：允许删除或调整选项，但不影响历史记录原值
- 基础选项修改规则：只影响后续新记录，历史记录保持原值
- 报表历史选项处理：历史旧名称按原值单独统计；统计金额为 0 的分类不展示
- 基础设置操作反馈：新增、编辑、删除成功后显示成功提示，并刷新当前列表
- 密码操作反馈：首次设置、修改成功后显示成功提示
- 备份操作反馈：导出备份、从备份恢复均显示成功或失败结果提示
- 安装包范围：仅 Windows 版
- 后续产出：将当前设计记录整理成正式需求文档
