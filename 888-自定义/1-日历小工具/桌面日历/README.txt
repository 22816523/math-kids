桌面浮动日历工具 - 使用说明
================================

一、环境要求
-----------
- Python 3.8 或更高版本
- Windows 操作系统

二、安装依赖
-----------
打开命令提示符（CMD）或 PowerShell，执行以下命令：

pip install PyQt5

如果安装速度慢，可使用国内镜像源：

pip install PyQt5 -i https://pypi.tuna.tsinghua.edu.cn/simple

三、运行程序
-----------
方式1：直接运行 Python 脚本
cd D:\文档\JTY\VS Code\0-工具\桌面日历
python calendar_app.py

方式2：双击运行（需先关联 .py 文件与 Python）

四、打包成 EXE 可执行文件
------------------------
1. 首先安装 PyInstaller：
pip install pyinstaller

2. 执行打包命令：
cd D:\文档\JTY\VS Code\0-工具\桌面日历
pyinstaller --onefile --windowed --name "桌面日历" --icon=NONE calendar_app.py

3. 打包完成后，可执行文件位于：
D:\文档\JTY\VS Code\0-工具\桌面日历\dist\桌面日历.exe

4. 可选：添加自定义图标
如果有 .ico 图标文件，可替换命令中的 --icon=NONE 为：
--icon=你的图标.ico

五、功能说明
-----------
1. 视图切换
   - 点击「本周」按钮显示当前周的7天
   - 点击「本月」按钮显示整月日历

2. 日期导航
   - 点击 ◀ 按钮切换到上一周/月
   - 点击 ▶ 按钮切换到下一周/月

3. 日期详情
   - 点击任意日期可查看详情面板
   - 显示农历日期、宜忌、冲煞信息

4. 休息日标记
   - 红色背景的日期为2026年法定节假日

5. 窗口操作
   - 拖动窗口任意位置可移动窗口
   - 点击右上角 × 按钮关闭程序
   - 窗口始终保持在最上层

六、2026年休息日清单
-------------------
- 元旦：1月1日-3日
- 春节：1月28日-2月3日
- 清明节：4月4日-6日
- 劳动节：5月1日-5日
- 端午节：5月31日-6月2日
- 中秋节+国庆节：9月27日-10月8日

七、常见问题
-----------
Q: 运行时提示找不到 PyQt5 模块？
A: 请确保已正确安装 PyQt5，执行 pip install PyQt5

Q: 打包后的 EXE 文件很大？
A: 这是正常现象，PyQt5 库较大，打包后约 30-50MB

Q: 如何修改休息日数据？
A: 编辑 calendar_app.py 文件中的 REST_DAYS_2026 变量

Q: 窗口无法拖动？
A: 请确保在窗口空白区域拖动，按钮区域点击会触发其他功能
